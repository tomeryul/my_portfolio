    parallel computations

    This project is a hybrid MPI, OpenMP, and CUDA application designed to perform parallel computations on large datasets. It reads input from a file, distributes the workload among multiple processes using MPI, and processes the data points in parallel on a GPU using CUDA. Each process computes a set of points based on proximity criteria, with the results being gathered and written to an output file. The program efficiently handles large-scale computations with concurrent processing and measures runtime performance.
